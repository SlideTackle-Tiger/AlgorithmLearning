# 栈
主要是模拟栈的应用
### 20 有效的括号
Deque 双端队列只用一头模拟栈，有两个实现类  
- ArrayDeque(基于数组，线程不安全，单机环境效率高) 
- inkedList(基于链表，多线程安全，插入删除效率高)

#### 算法详解
Deque + Map （辅助map 用于配对括号对 key 右括号，value 左括号）  
遍历S对于每一个字符首先判断是否是map的key(右括号)
右括号那么查看栈顶元素是否为匹配的左括号，不是则return false
左括号则压入栈中  
最终判断是否满足的条件是如果栈为空，则表示所有括号都匹配上了。  

### 最小栈
题目要求设计一种栈的数据结构，支持常数时间内找到最小元素。
- 两个栈用于实现，一个栈用于保存数据，另一个栈用于保存最小值。
- push操作，首先判断最小栈是否为空 ｜｜ 当前元素小于等于最小栈栈顶元素，满足则插入，不满足则只插入数据栈。
- pop操作，首先判断数据栈元素是否为空，如果空则抛出异常，再比较最小栈栈顶元素与数据栈栈顶元素，如果等于则弹出最小栈与数据栈顶元素，否则只弹出数据栈元素

### 字符串解码
题目要求给定一个经过编码的字符串，返回解码后的字符串，编码规则为：k[encoded_string] =》 表示encode_string重复k次。  
#### 算法详解
- 首先维护两个栈用于存储数字、字符。currentString 用于存储当前解码的字符串，k用于存储当前倍数。
- 遍历给定编码数组，对于任意一个字符有以下四种情况：  
  - 当前字符是数字，那么我们计算翻倍的倍数。 Character.isDigit()判断是否为数字， k = k * 10 + （c-'0'）来判断倍数。
  - 如果当前字符是[ 左括号，那么我们将字符串与数字压入栈中。并重置currentString，k。
  - 如果当前字符是] 右括号，那么我们进行解码操作，利用StringBuilder来操作数字栈，字符栈的栈顶元素，将处理结果传给currentString。
  - 如果当前字符是字母，那么我们直接将其放到currentString中。
- 最后返回currentString。
